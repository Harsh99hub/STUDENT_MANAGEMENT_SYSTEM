# STUDENT_MANAGEMENT_SYSTEM
🎓 Student Management System

A console-based Student Management System developed using Python that helps manage student records efficiently. This project allows users to perform CRUD operations such as adding, viewing, searching, updating, and deleting student records.

**📌 Project Description**

The Student Management System is a beginner-friendly Python project designed to manage student information using a simple command-line interface.

This project demonstrates:

Python fundamentals
Functions and modular programming
Data management concept

It is ideal for:

📚 BCA / MCA students
💻 Python beginners
🛠️ Mini projects
🎯 Academic submissions

**✨ Features**

✅ Add New Student
✅ View All Student Records
✅ Search Student by ID or Name
✅ Update Student Details
✅ Delete Student Record
✅ Store Data using:

✅ Simple Console Interface
✅ Easy to Understand Code Structure
